# SubletUC

## Overview
SubletUC is a specialized housing search application designed to help students find temporary housing, sublets, and roommates for the semester. It addresses the specific needs of the university community by providing a secure, reliable, and user-friendly platform to connect students looking for housing with those who have space available.

## Features

*   **User Authentication**: Secure sign-in and account creation.
*   **Sublet Search**: Browse a grid of listings with powerful filtering and sorting options (Rent, Bedrooms, Housing Type, etc.).
*   **Interactive Map**: Toggle between grid view and a live map view to see listings in relation to campus.
*   **Post a Listing**: Easy-to-use interface for students to list their apartments or rooms for sublet.
*   **Saved Listings**: Bookmark interesting properties to review later.
*   **Profile Management**: Manage personal information and listing history.
*   **Responsive Design**: Built to work seamlessly on desktops, tablets, and mobile devices.

## Technology Stack

### Frontend
*   **Framework**: React (TypeScript)
*   **Styling**: Tailwind CSS
*   **Icons**: Lucide React

### Backend (Architecture)
*   **Framework**: .NET 8 Web API
*   **Database**: Microsoft SQL Server
*   **ORM**: Entity Framework Core

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm start
    ```
    **Run Back End Development Server**
    ```bash 
    dotnet run
    ```
